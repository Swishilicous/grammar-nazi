# grammar-nazi
A powercord plugin that forces you to use *obnoxiously* good grammar.

## Features
* Capitalize the first letter of every message.
* Always add punctuation at the end.
* Simple Question Detection.
* Spellcheck Node Module.
* Toggleable Options.
